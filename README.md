# Zunary

**Zunary — agendamentos simples para negócios organizados.**

Zunary é um SaaS simples de agendamento online para negócios locais.  
O objetivo do MVP é permitir que uma empresa cadastre seus serviços, defina disponibilidade e receba agendamentos por meio de uma página pública.

---

## Visão geral

A Zunary foi criada para resolver um problema comum de pequenos negócios: organizar agendamentos de forma simples, sem depender de planilhas, mensagens soltas ou processos manuais.

Fluxo principal:

```txt
Usuário cria conta
→ cria/vincula empresa
→ escolhe um plano
→ cadastra serviços
→ define disponibilidade
→ cliente agenda pela página pública
→ dono acompanha os agendamentos no painel
